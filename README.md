# Context Switcher

컨텍스트 전환을 위한 듀얼 에디터 노트 앱. Electron 기반의 Windows 포터블 실행 파일로 배포됩니다.

## 주요 기능

- **듀얼 에디터**: 두 개의 텍스트 에디터를 나란히 열어 동시에 비교하거나 편집
- **폴더 탭**: 홈 / 업무 / 아이디어 / 메모 4개의 폴더로 노트 분류
- **드래그 앤 드롭**: 사이드바 아이템을 에디터로 드래그하여 열기
- **자동 저장**: 입력 후 500ms 디바운스로 자동 저장
- **아카이브**: 체크박스로 완료된 아이템을 아카이브로 이동, 복원 가능
- **인라인 이름 변경**: 아이템 또는 에디터 제목을 더블클릭하여 이름 변경
- **다크/라이트 테마**: 시스템 설정을 따라 자동 전환
- **탭 이모지 커스터마이징**: 폴더 탭을 더블클릭하여 이모지 변경

## 데이터 저장 위치

| 환경 | 경로 |
|------|------|
| 패키징된 앱 | `C:\Users\{user}\AppData\Roaming\ContextSwitcher\notes\` |
| 개발 모드 | 프로젝트 루트의 `notes\` 폴더 |

각 폴더(`home`, `work`, `idea`, `memo`)에 `.txt` 파일로 저장됩니다.

## 사용 방법

**아이템 생성**: 사이드바 상단 입력창에 제목 입력 후 `Enter`

**에디터에 열기**
- 좌클릭 → 에디터 1에 열기
- 우클릭 → 에디터 2에 열기
- 드래그 앤 드롭 → 원하는 에디터에 열기

**아이템 이름 변경**: 아이템을 더블클릭하여 인라인 편집

**아카이브**: 아이템 체크박스 클릭 → 아카이브로 이동
**아카이브 복원**: 하단 "아카이브 보기" → 복원 버튼 클릭 또는 에디터로 드래그

## 개발 환경 실행

```bash
npm install
npm start
```

## 빌드

Windows 포터블 실행 파일(`ContextSwitcher.exe`) 생성:

```bash
npm run build
```

빌드 결과물은 `dist/` 폴더에 생성됩니다.
